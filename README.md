# 삐뽀(PPIBBO) - 지능형 실시간 AI 모바일 관제 서비스 🚨

**대학가 치안 사각지대 및 생활 불법행위 해소를 위한 YOLOv8 기반 통합 관제 어플리케이션**

## 1. 프로젝트 개요
[cite_start]대학가 및 원룸촌 등 관리 사각지대에서 발생하는 흡연, 무단 투기, 전동 킥보드 위반 등 생활형 불법행위를 AI로 실시간 감지하여 관리자에게 알림을 제공하는 시스템입니다. [cite: 375]

## 2. 핵심 기능
* [cite_start]**AI 실시간 행동 감지**: YOLOv8-Pose를 활용해 코와 손목 사이의 유클리드 거리를 측정하여 흡연 동작을 판별하고, 물체와의 벡터 거리 변화로 무단 투기를 감지합니다. [cite: 395, 400]
* [cite_start]**프라이버시 보호 (기본 패시브)**: 행인의 초상권 보호를 위해 평상시 모니터링 화면에서는 얼굴 영역을 실시간으로 블러(Blur) 처리합니다. [cite: 391, 520]
* [cite_start]**모바일 관제 및 알림**: FastAPI 기반 백엔드를 구축하여 위반 상황 발생 시 관리자의 모바일 기기로 즉각적인 푸시 알림과 시각적 경고(붉은색 테두리 점멸)를 전송합니다. [cite: 378, 546]
* [cite_start]**오디오 피드백**: 위반 감지 시 앱 알림과 동시에 현장에 TTS(Text-to-Speech) 안내 방송을 자동으로 송출합니다. [cite: 455]

## 3. 기술 스택 (Tech Stack)
* [cite_start]**AI & Vision**: Python, YOLOv8 (Pose/Detect), OpenCV [cite: 394, 399, 410]
* [cite_start]**Backend**: FastAPI, ngrok (네트워크 터널링) [cite: 381, 413]
* [cite_start]**Frontend**: HTML/CSS/JS (Hybrid App), Chart.js [cite: 382, 423]

## 4. 성과 및 성장 포인트 (Learning By Doing)
* **주도적 역할 수행**: 작년 프로젝트의 수동적인 팀원 역할에서 벗어나, 이번 프로젝트에서는 기술 설계부터 시스템 통합까지 전 과정을 주도적으로 이끌었습니다.
* [cite_start]**실전 문제 해결**: 로컬 서버의 외부 접속 한계를 `ngrok` 터널링으로 해결하고, 프라이버시 보호를 위한 조건부 블러 처리 로직을 직접 고안하여 구현했습니다. [cite: 413]
* [cite_start]**정확도 확보**: 지속적인 튜닝을 통해 위반 행위 감지 정확도 97%~99% 수준을 확보했습니다. [cite: 528, 571]
